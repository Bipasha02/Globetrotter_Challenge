# Globetrotter_Challenge

A full-stack web app for guessing famous destinations based on cryptic clues. Built with React (frontend) and Node.js/Express (backend).

## Setup
1. **Backend**:
   - `cd backend`
   - `npm install`
   - `npm start`
2. **Frontend**:
   - `cd frontend`
   - `npm install`
   - `npm start`

## Tech Stack
- **Frontend**: React, Axios, React-Confetti
- **Backend**: Node.js, Express

## Features
- Random clues with multiple-choice answers
- Confetti animation for correct answers, sad feedback for incorrect
- Score tracking
- "Challenge a Friend" via WhatsApp
